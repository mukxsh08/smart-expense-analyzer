💰 Smart Expense Analyzer

An AI-powered web application to track, analyze, and visualize personal expenses.
Built with a modern full-stack architecture for real-time insights and better financial decisions.

🚀 Features
📊 Add and manage daily expenses
🧾 Categorize expenses (Food, Travel, Bills, etc.)
📈 Visualize spending with charts
💡 Get insights on spending patterns
⚡ Fast API-based backend
🌐 Simple and responsive frontend

⚙️ How to Run Locally
1️⃣ Clone the repo
git clone https://github.com/your-username/smart-expense-analyzer.git
cd smart-expense-analyzer
2️⃣ Setup Backend
cd backend
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
3️⃣ Run Backend
uvicorn main:app --reload

👉 Backend runs at:
http://127.0.0.1:8000

4️⃣ Run Frontend

Just open:

frontend/index.html

OR use Live Server in VS Code.
